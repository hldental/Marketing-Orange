# Consult Marketing Pages - Orange / Black Version

This package includes two separate Next.js landing pages with updated Huntline Dental Group branding and logo.

Routes:
- /surgery-consult
- /general-consult

Before you deploy, replace these placeholders:
- REPLACE_WITH_SURGERY_FORM_ID
- REPLACE_WITH_GENERAL_FORM_ID

Use full Formspree URLs in this format:
- https://formspree.io/f/your_surgery_id
- https://formspree.io/f/your_general_id
